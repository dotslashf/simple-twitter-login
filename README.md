# Simple Twitter Login

Simple twitter login with Express JS and HTML

## Description

Easily login your twitter bot account to primary account.

## Getting Started

### Installing Dependencies

    npm install

    or

    yarn install

### Running

- Rename file `.env.example` to `.env`
- Copy your credentials from twitter developer accounts to `.env`
- Run server with:

  ```
  npm run dev

  or

  yarn dev
  ```

- Open localhost:3000 or preferred PORT
- Click on login button
- Your bot twitter credentials will be written to `.env`
